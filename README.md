# Login Attempt Alert System

## Description
This is a Python-based security project that detects multiple failed login
attempts and raises an alert to indicate a possible brute-force attack.

## Features
- Detects failed login attempts
- Raises alert after multiple failures
- Beginner-friendly implementation

## Technologies Used
- Python

## How to Run
1. Clone or download the repository
2. Open terminal in the project folder
3. Run the command: `python login_alert.py`
4. Enter login credentials to test
